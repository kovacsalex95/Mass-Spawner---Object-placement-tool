Preparations

This extension needs a GameObject in your scene in order to work, and a MassSpawner component in it.
Before we get started:
  • Create an empty GameObject
  • Add a MassSpawner component to it

Editor Panel

We just created a MassSpawner instance, so we can work with it.
But before that, we need to understand what we seeing:
  • Heightmap view: Here you can see the selected area's height and steepness graph, or a stylized 3d view.
    Currently this is empty because we dont have heightmap yet. After you create a heightmap, this panel gonna look like this:
  • #1: Here you can switch from the 3 modes:
    • Heightmap
    • Steepness graph
    • Stylized view
  • #2: Here you can select a configuration page (#3):
    • World: Here you can define your world's boundary.
    • Heightmap: Here you can define from wich layers you want to sample the height data, and wich layer you want to ignore. You can change the heightmap resolution as well. More information
    • Layers: Here you can define the layers and their rules. This defines wich prefab gonna be spawned, where, and how. More information
    • Color groups: Here you can define coloring groups. You can assign this groups to a layer's prefab. More information
